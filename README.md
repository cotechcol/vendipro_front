# POS System - Frontend

Interfaz web para sistema de punto de venta con Vue 3, TypeScript y Tailwind CSS.

## Requisitos

- Node.js 18+
- Backend corriendo en `http://localhost:3000`

## Instalación

```bash
npm install
npm run dev
```

La aplicación inicia en `http://localhost:5173`.

## Credenciales por defecto

- **Email:** admin@pos.local
- **Contraseña:** admin123

## Flujo de uso

1. Iniciar sesión
2. Abrir caja (menú Caja)
3. Registrar productos y categorías (Admin)
4. Realizar ventas en Punto de Venta
5. Imprimir ticket
6. Cerrar caja al final del turno
7. Consultar reportes

## Pantallas

- **Dashboard:** resumen del día
- **POS:** ventas rápidas con carrito
- **Productos/Categorías:** catálogo
- **Inventario:** movimientos y alertas de stock
- **Clientes/Proveedores:** datos maestros
- **Compras:** registro de entradas de mercancía
- **Caja:** apertura y cierre
- **Reportes:** ventas, inventario y rentabilidad
- **Usuarios/Configuración:** solo Admin
